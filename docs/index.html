Global Hotkey

Instantly summon Gemini Desktop from any app using Alt + Space (or Cmd + Opt + Space on Mac). It auto-focuses the prompt so you can start typing immediately.

Smart Memory Optimized

Built with an optimized Electron core. Uses background garbage collection to keep the RAM footprint light. Remembers your exact screen coordinates and custom window size.

Native Voice

Full integration with your OS microphone permissions for seamless Voice-to-Text dictation.

A clean, native wrapper that perfectly integrates with your OS.
