Gemini Desktop

Experience Google Gemini outside the browser. Native, fast, and packed with power user features.

🔒 Direct, secure webview. No injected scripts. Your Google session stays strictly local.
Download Gemini Desktop ☕ Support the Project

Global Hotkey

Instantly summon Gemini from any app using Alt + Space (or Cmd + Opt + Space on Mac). It auto-focuses the prompt so you can start typing immediately.

Smart Memory Optimized

Built with an optimized Electron core. Uses background garbage collection to keep the RAM footprint light. Remembers your exact screen coordinates and custom window size.

Native Voice

Full integration with your OS microphone permissions for seamless Voice-to-Text dictation.
